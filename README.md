# Google-Earth-Engine-Scripts-Qualidade-do-ar-no-RS-Brasil
Este repositório reúne scripts em JavaScript desenvolvidos para a coleta de dados atmosféricos relacionados à qualidade do ar no estado do Rio Grande do Sul (Brasil). Poluentes analisados: material particulado (PM₂,₅), dióxido de nitrogênio (NO₂), dióxido de enxofre (SO₂), ozônio (O₃), em µg/m³; e monóxido de carbono (CO) em ppm.
